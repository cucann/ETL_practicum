## Цель работы

Получить практические навыки работы с библиотекой Dask для построения базовых ETL-конвейеров (Extract, Transform, Load) при обработке больших массивов данных, не помещающихся в оперативную память. ## Проблемы и их решение

| Проблема | Решение |
|----------|---------|
| Ошибка типов данных при чтении CSV | Явное указание dtype для всех 51 столбца |
| Некорректные форматы дат (например, `20140831` и `00/15/2023`) | Использование `errors='coerce'` для преобразования некорректных значений в NaT |
| Ошибка `'int' object has no attribute 'compute'` | Использование `df.shape[0]` вместо `len(df.index)` |
| Постоянные вылеты браузера (конкретной вкладки) из-за недостатка оперативной памяти | Перезапуск вкладки, очень быстрый запуск всех блоков кода, пока память не успела забиться |
